--- layout: post title: Installing Debian 5 (Lenny) on Microsoft Virtual Server 2005 R2 SP1 --- If you run into a kernel panic installing Debian 5 (Lenny) on Microsoft Virtual Server 2005 R2 SP1 (32 bit host and client), try this:
  1. Launch the Debian CD.
  2. Press TAB on the Install option to edit the command line.
  3. Change 
    vga=normal
    to 
    vga=791
  4. Add the following: 
    noapic nolapic noreplace-paravirt
  5. Install as normal.
