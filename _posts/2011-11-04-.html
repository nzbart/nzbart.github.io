--- layout: post title: '' --- First clear your existing rules, if any (Home > Conditional Formatting > Clear Rules)

Then select the area you want to have the conditional formatting and not which row the active cell is in. Suppose it is in row 2...
Home > Conditional Formatting > New Rule > Use formula to determine which cells to format > Format values where this formula is true: =$B2="No" > Format > set the colour you desire.
The 2 in the formula should be the row number containing the active cell.
Note there is no $ before the 2, which means the row number adjusts for the other rows
