--- layout: post title: Get a list of missing updates on Server Core --- If you want to get a list of Windows Updates that haven't been installed on Windows Core (or any machine for that matter where you may want to use the command line), the following PowerShell one-liner will work: 

(New-Object -ComObject Microsoft.Update.Searcher).Search("IsInstalled=0 and Type='Software'").Updates | select LastDeploymentChangeTime, Title | ft -AutoSize
